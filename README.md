Life game simulation on fictitious 1 bit MPP machine simulator.

## usage

1. Compile C++ source.

python3 setup.py build_ext --inplace

2. Execute mpp_lifegame.py with life game pattern file.

python3 mpp_lifegame.py pattern/puffer_train.life 

## Requirement

* Python3
* PyGame
* Pillow
